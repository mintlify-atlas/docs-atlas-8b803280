## Get Welcome Message

<CodeGroup>
```bash curl
curl -X GET https://your-mcrit-instance.com/ \
  -H "apitoken: YOUR_API_TOKEN"
```
</CodeGroup>

<ResponseExample>
```json Response
{
  "status": "successful",
  "data": {
    "message": "Welcome to MCRIT"
  }
}
```
</ResponseExample>

Returns a welcome message to confirm the API is accessible.

## Get System Status

<CodeGroup>
```bash curl
curl -X GET https://your-mcrit-instance.com/status \
  -H "apitoken: YOUR_API_TOKEN"
```

```bash With PicHash
curl -X GET https://your-mcrit-instance.com/status?with_pichash=true \
  -H "apitoken: YOUR_API_TOKEN"
```
</CodeGroup>

<ParamField query="with_pichash" type="boolean" default="true">
  Include PicHash statistics in the response. Set to `false` to exclude.
</ParamField>

<ResponseExample>
```json Response
{
  "status": "successful",
  "data": {
    "status": {
      "db_state": "operational",
      "db_timestamp": "2026-03-03T10:30:00Z",
      "storage_type": "mongodb",
      "num_bands": 128,
      "num_samples": 1234,
      "num_families": 56,
      "num_functions": 45678,
      "num_pichashes": 40000
    }
  }
}
```
</ResponseExample>

Returns comprehensive status information about the MCRIT instance, including database state and statistics.

<Expandable title="Response Fields">
  <ResponseField name="status" type="string">
    Request status ("successful" or "failed")
  </ResponseField>
  
  <ResponseField name="data.status" type="object">
    System status information
    
    <Expandable title="Status Object">
      <ResponseField name="db_state" type="string">
        Current state of the database
      </ResponseField>
      
      <ResponseField name="db_timestamp" type="string">
        Timestamp of last database update
      </ResponseField>
      
      <ResponseField name="storage_type" type="string">
        Type of storage backend (e.g., "mongodb")
      </ResponseField>
      
      <ResponseField name="num_bands" type="integer">
        Number of MinHash bands in the index
      </ResponseField>
      
      <ResponseField name="num_samples" type="integer">
        Total number of samples stored
      </ResponseField>
      
      <ResponseField name="num_families" type="integer">
        Total number of malware families
      </ResponseField>
      
      <ResponseField name="num_functions" type="integer">
        Total number of functions indexed
      </ResponseField>
      
      <ResponseField name="num_pichashes" type="integer">
        Total number of PicHashes computed
      </ResponseField>
    </Expandable>
  </ResponseField>
</Expandable>

## Search Families

<CodeGroup>
```bash curl
curl -X GET "https://your-mcrit-instance.com/search/families?query=emotet&limit=10" \
  -H "apitoken: YOUR_API_TOKEN"
```

```bash With Sorting
curl -X GET "https://your-mcrit-instance.com/search/families?query=emotet&sort_by=family_name&is_ascending=true" \
  -H "apitoken: YOUR_API_TOKEN"
```
</CodeGroup>

Search for malware families by name or other attributes.

<ParamField query="query" type="string" required>
  Search term to match against family fields
</ParamField>

<ParamField query="limit" type="integer">
  Maximum number of results to return
</ParamField>

<ParamField query="cursor" type="string">
  Pagination cursor from previous search results
</ParamField>

<ParamField query="sort_by" type="string">
  Field name to sort results by
</ParamField>

<ParamField query="is_ascending" type="boolean" default="true">
  Sort direction (true for ascending, false for descending)
</ParamField>

<ResponseExample>
```json Response
{
  "status": "successful",
  "data": {
    "search_results": {
      "1": {
        "family_id": 1,
        "family_name": "emotet",
        "num_samples": 42,
        "is_library": false
      }
    },
    "cursor": {
      "forward": "next_cursor_token",
      "backward": null
    }
  }
}
```
</ResponseExample>

## Database Operations

### Respawn Database

<Warning>
  This operation will delete all data and create a new empty MCRIT instance. Use with extreme caution.
</Warning>

<CodeGroup>
```bash curl
curl -X POST https://your-mcrit-instance.com/respawn \
  -H "apitoken: YOUR_API_TOKEN"
```
</CodeGroup>

<ResponseExample>
```json Response
{
  "status": "successful",
  "data": {
    "message": "Successfully performed reset of MCRIT instance."
  }
}
```
</ResponseExample>

Drops all storage and sets up a new empty MCRIT instance.
