---
title: "List Families"
description: "Retrieve a list of all families with pagination support"
---

## Endpoint

```
GET /families
```

Retrieve a paginated list of all families in the MCRIT database. This endpoint returns an overview of all families including their metadata and statistics.

## Query Parameters

<ParamField query="start" type="integer" default="0">
  The starting index for pagination. Use this to skip a number of families from the beginning of the results.
</ParamField>

<ParamField query="limit" type="integer" default="0">
  Maximum number of families to return. When set to 0, all families from the start index onwards are returned.
</ParamField>

## Response

<ResponseField name="status" type="string" required>
  Status of the request. Returns `"successful"` on success.
</ResponseField>

<ResponseField name="data" type="object" required>
  Dictionary of family entries keyed by family ID.
  
  <Expandable title="Family Entry Object">
    <ResponseField name="family_id" type="integer" required>
      Unique identifier for the family.
    </ResponseField>
    
    <ResponseField name="family_name" type="string" required>
      Name of the family.
    </ResponseField>
    
    <ResponseField name="num_samples" type="integer" required>
      Total number of samples in this family.
    </ResponseField>
    
    <ResponseField name="num_functions" type="integer" required>
      Total number of functions across all samples in this family.
    </ResponseField>
    
    <ResponseField name="num_library_samples" type="integer" required>
      Number of samples in this family that are marked as libraries.
    </ResponseField>
  </Expandable>
</ResponseField>

## Examples

### List all families

```bash
curl -X GET "http://localhost:8000/families"
```
