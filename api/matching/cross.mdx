## Get Cross Sample Matches

Performs cross-matching between multiple samples, finding similarities across all provided sample IDs.

<ParamField path="sample_ids" type="string" required>
  Comma-separated list of sample IDs to cross-match (e.g., "123,456,789")
</ParamField>

<ParamField query="minhash_score" type="integer" default="50">
  Minimum MinHash similarity score threshold (0-100)
</ParamField>

<ParamField query="band_matches_required" type="integer" default="0">
  Minimum number of band matches required
</ParamField>

<ParamField query="pichash_size" type="integer" default="8">
  Size of PicHash for matching (must be >= 0)
</ParamField>

<ParamField query="force_recalculation" type="boolean" default="false">
  Force recalculation of matches instead of using cached results
</ParamField>

<ParamField query="sample_group_only" type="boolean" default="false">
  Only match against samples in the same group
</ParamField>

### Request

```bash
curl -X GET "https://your-server.com/matches/sample/cross/123,456,789?minhash_score=70" \
  -H "apitoken: YOUR_API_TOKEN"
```

### Use Cases

Cross-matching is useful for:
- Identifying common code patterns across multiple samples
- Finding shared library code between binaries
- Analyzing malware families for shared components
- Detecting code reuse across different versions
